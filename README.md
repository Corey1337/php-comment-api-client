## Example API Client for comments
This client is built upon PSR-7, PSR-17 and PSR-18 standards

## Requirements
- PHP >= 8.5
- Composer >= 2.9
- PSR-17 Factory and PSR-18 Client implementation (Guzzle, Nyholm\Psr7, Symfony HTTP Client)

## Installation
- Add 
```
"repositories": [
    {
        "type": "vcs",
        "url": "https://github.com/Corey1337/php-comment-api-client"
    }
]
```
- Execute `composer require corey/php-comment-api-client`
